Termos de Uso

Este documento estabelece os Termos de Uso para o Takuya, um bot desenvolvido para servidores de Discord com foco em RPG, RP e interações personalizadas. A utilização do Takuya implica na aceitação das condições abaixo.

- É proibido utilizar o bot para atividades que violem os Termos de Serviço do Discord.

Privacidade e Dados

- O Takuya coleta informações limitadas para funcionamento, como nomes de usuários, mensagens e comandos.
- Os dados coletados não são vendidos ou compartilhados com terceiros, exceto quando exigido por lei.
- Consulte a Política de Privacidade para mais detalhes.

Restrições de Uso

- Não é permitido modificar, redistribuir ou usar o bot para fins comerciais sem autorização.
- Qualquer abuso ou uso indevido pode resultar na remoção do Takuya do servidor.

Atualizações nos Termos

Estes Termos de Uso podem ser atualizados periodicamente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados.
